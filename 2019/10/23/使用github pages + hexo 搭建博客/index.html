使用github pages + hexo 搭建博客

安装并部署博客

  1. 前往github.com创建一个账户
  2. 创建一个名为username .github.io 的新存储库，其中username是您在GitHub上的用户名（或组织名称）
  3. 转到要存储项目的文件夹，然后克隆新的存储库：

    git clone https://github.com/ 用户名 / 用户名 .github.io

  4. 下载并安装nodeJs
    选一个即可，下载后安装时一路默认next(安装路径可以自选)
  5. 在终端中输入

    npm install hexo-cli -g
    npm install hexo –save
    【如果不行可能是权限不够，自觉加sudo】

  6. 在你本地创建一个文件夹用于存放你的博客内容，例如名为 blog 的文件夹
  7. 通过终端进入到blog的目录中
    ps:你可能需要用到 pwd、 ls、cd

    hexo init 等待框架安装完成

  8. 在_config.yml,进行基础配置
    拉到最下面找到 deploy 添加以下信息

    type 为上传的方式
    repo 为上传的仓库地址
    branch 为上传所在的分支

    ps:_config.yml文档修改所有信息都要注意，冒号后面一定要有一个空格！冒号后面一定要有一个空格！冒号后面一定要有一个空格！重要的事情说三遍！
  9. 配置完成后保存，在终端上打开到当前的blog目录，输入

    npm instal hexo-deployer-git –save

  10. 现在将你的代码推到网上去：依次输入

    hexo clean
    hexo g
    hexo d

    clean     Remove generated files and cache.
config    Get or set configurations.
deploy    Deploy your website.

发表文章

  1. 你可以在使用你喜欢的markdown编辑器进行编辑你的.md文档
    但请最后把你的编辑好的文档放入blog\source_posts文件夹里面
    当然你也可以直接在这个目录中直接编辑.md文档

  2. 写完保存之后，输入

    hexo g -d

    一键上传

    等待系统把文件上传到远程仓库中(或许很快或许很慢)

  3. 然后打开浏览器输入你的 username.github.io，你就可以看到你的博客啦。
